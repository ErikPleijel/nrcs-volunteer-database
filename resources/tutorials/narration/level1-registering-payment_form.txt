Payments work the same way — open Payment Records, press Add, and select the person. <break time="0.6s" /> The payment form has a little more to it. <break time="0.6s" /> First, choose the Fee. The list automatically shows the right fees for this person — member fees for members, and volunteer fees for volunteers. <break time="0.6s" /> For volunteers, you can tick ID Card to include one, and its fee is added to the total. Members do not receive ID cards here. <break time="0.7s" /> Set the Payment Date. By default it is today — but if the person already has a valid membership, the date shifts automatically to the day after that one expires, so the two do not overlap. The Current Payment panel above shows their existing cover, so you never charge twice. <break time="0.7s" /> Add a Reference if you have one. <break time="0.6s" /> Finally, check the Payment Summary — it adds up the fee and any ID card fee — then press Register Payment.
