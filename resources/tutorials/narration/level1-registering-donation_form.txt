Donations work the same way — open Donation Records, press Add, and select the person. <break time="0.6s" /> Set the Donation Date. <break time="0.6s" /> Then decide the type. For a cash gift, leave the box unticked and enter the Amount. For goods — such as food, blankets, or supplies — tick In-Kind Donation, then name the item and how many were given. <break time="0.7s" /> If the donor wishes to stay unnamed, tick Anonymous donation. <break time="0.6s" /> Add a Reference and a Purpose if you have them. <break time="0.6s" /> Then press Create Donation to store the record.
